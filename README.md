# 🏎️ Racing Web Game

Juego de carreras web 3D multijugador hecho con Babylon.js, TypeScript y Vite.

## 🎮 Características

- ✅ **Gráficos 3D realistas** con Babylon.js
- ✅ **Multijugador P2P** con PeerJS
- ✅ **7 autos diferentes** para elegir
- ✅ **Pista extendida** de ~12 km (~2 min por vuelta)
- ✅ **Audio del motor** sintetizado en tiempo real
- ✅ **Música phonk/drift** de fondo
- ✅ **Física arcade** divertida
- ✅ **Gradas rojas** alrededor del circuito
- ✅ **Montañas con nieve** en el horizonte
- ✅ **Árboles y vegetación**
- ✅ **Sistema de vueltas** (3 vueltas por carrera)

## 🚀 Instalación

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev

# Compilar para producción
npm run build
```

## 🎯 Controles

- **W / ↑** - Acelerar
- **S / ↓** - Reversa
- **A / ←** - Girar izquierda
- **D / →** - Girar derecha
- **Espacio** - Freno
- **1-4** - Cambiar cámara

## 🔧 Tecnologías

- **Babylon.js** - Motor 3D
- **TypeScript** - Lenguaje
- **Vite** - Build tool
- **PeerJS** - Multijugador P2P
- **Web Audio API** - Audio sintetizado
